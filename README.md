# About
go-wav is easy way to use wav file package for the Go programming language. 

# Commands
* [wavparser](https://github.com/takuyaohashi/go-wav/tree/master/cmd/wavparser)

## How to install
```bash
$ go get github.com/takuyaohashi/go-wav/cmd/...
```
